DogFlix é uma aplicação web para realizar cadastros de séries.

<h2>Aplicação</h2>

-Comtem uma tela de login <br>
-CRUD das Séries <br>
-Tela para exibir todas as séries cadastradas no sistema

<h2>Tecnologias usadas</h2>

-NetBeans <br>
-Java Maven 17
